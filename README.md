<h3>ℹ️ Como usar</h3>

    # Clone esse repositório
    $ git clone https://github.com/Diegooliveyra/dogflix-Imersao-Alura.git
    
    # Instalar dependências
    $ npm install
    
    # Executar
    $ npm start
